# 🍼 ベビーカーNavi — 東京鉄道移動支援アプリ

ベビーカーで東京の電車移動をもっとラクに。  
出発駅・到着駅・時刻を入れるだけで「何号車に乗ればエレベーターに近いか」を一発表示します。

---

## セットアップ

```bash
# 依存パッケージのインストール
cd baby_stroller_nav
npm install

# 開発サーバー起動
npm run dev
# → http://localhost:5173 で開きます

# ビルド（本番用）
npm run build
```

### システム要件

- Node.js 18 以上
- npm 8 以上

## ベビースコアの計算ロジック

| 項目 | 配点 | 基準 |
|------|------|------|
| 🛗 エレベーター | 0〜40点 | 乗降両方にエレベーター情報あり → 高得点 |
| 🚶 ホーム移動 | 0〜20点 | ホーム内移動が少ない → 高得点 |
| 🔄 乗換 | 0〜20点 | 乗換なし→20点 / 1回→10点 / 2回以上→0点 |
| ⏱ 所要時間 | 0〜20点 | 短いほど高得点（60分超は-5点） |

**合計 100点満点**  
- 85点以上：最適 🟢  
- 70〜84点：おすすめ 🔵  
- 69点以下：ふつう 🟡

## 使用データの前提

現在 **モックデータ** を使用しています。以下のデータはすべて参考値です：

- 時刻表: 固定間隔（4〜10分）で発車すると仮定して生成
- 号車番号・エレベーター位置: 実際の駅設備を参考にした概算値
- 乗換時間: 一律 5 分で計算

### 本番データへの差し替えポイント

| ファイル | 差し替え内容 |
|----------|-------------|
| `src/logic/routeFinder.ts` の `findRoutes()` | 公共交通オープンデータ（ODPT）経路 API を呼び出す |
| `src/data/mockRoutes.ts` | ODPT 駅設備情報 API（バリアフリー情報）でエレベーター位置を取得 |
| `src/data/stations.ts` | ODPT 駅情報 API から全駅マスター取得 |

**推奨 API:**
- [公共交通オープンデータセンター (ODPT)](https://developer.odpt.org/) — 時刻表・運行情報
- [東京都公共交通オープンデータ](https://tokyochallenge.odpt.org/) — 駅設備・バリアフリー情報
